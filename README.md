# How to work with git


## 0. Check git version

``` git --version```

## 1. Init repository

``` git init ```

## 2. Add GitHub repository

``` git remote add origin < SSH LINK>```


## 3. Add file 

``` git add <FILENAME>```

## 4. Check status

``` git status```

## 5. Save changes (commit changes).

```git commit -m "SOME MESSAGE"```

## 6. Add changes to github

```git push origin master```
